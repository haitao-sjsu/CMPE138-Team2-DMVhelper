第1节：加州驾驶员执照（The California Driver’s License）

拥有加州驾照，您才被允许在公共道路上驾车。

 

在获得加州驾照之前，您必须根据本手册中的信息通过知识测试（笔试）（knowledge test）和路考（behind-the-wheel drive tests）。

 

您必须持有正确的驾照才能驾驶您的车型。大多数人需要非商业 C 级驾照（noncommercial Class C driver’s license）。要驾驶商用车辆、摩托车或其他类型的车辆，您必须持有相应类型的驾照。

 

欲了解有关 C 级驾照所涵盖的车辆信息，请访问 dmv.ca.gov/dl。

 

驾照相关名词一览（Card Designations）

REAL ID驾驶执照（REAL ID Driver’s License）

自 2025 年 5 月起，您的驾照或身份证件（ID）必须符合 REAL ID 标准，才能用于以下用途：

l 登上国内航班的飞机。

l 进入军事基地。

l 进入大多数联邦设施。

欲了解有关申请 REAL ID 的更多信息，请访问 dmv.ca.gov/realid。

 

无证居民的驾照（Driver’s Licenses for Undocumented Residents）

加州为所有居民提供驾照，无论其移民身份如何。欲了解更多信息，请访问 dmv.ca.gov/dl。

 

器官和组织捐献者（Organ and Tissue Donor）

有关信息，请访问 donateLIFEcalifornia.org。

 

退伍军人（Veterans）

欲了解更多有关退伍军人称号的要求和好处，请访问 dmv.ca.gov/veterans。

 

ID卡（ID Cards）

ID卡是为任何年龄的符合条件者颁发的身份证明。它们不是您开车的许可证。要获得ID卡，您必须提供身份证件（identity document）、居住证明文件（residency documents）和社会保险号（social security number）。

 

请访问 dmv.ca.gov/id-cards，填写ID卡申请表，并了解当前可接受的文件以及减费、免费或老年身份证的信息。
